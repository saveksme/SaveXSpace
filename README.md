# SaveXSpace

Быстрый, удобный и современный VPN-клиент для Windows и Android.

Построен на базе ClashMeta. Без рекламы, без лишнего.

## Возможности

- Глобальный и автоматический режимы маршрутизации
- Импорт подписок по URL, QR-коду или файлу
- Быстрый импорт через Ctrl+V (вставка URL из буфера обмена)
- Обход российских сайтов в глобальном режиме (трафик на .ru домены идёт напрямую)
- Тёмная тема с настраиваемыми акцентными цветами
- Информация о подписке: статус, срок действия, трафик
- Горячие клавиши для управления
- Поддержка WebDAV для синхронизации
- Системный трей с быстрым управлением

## Платформы

- Windows 10/11 (x64)
- Android 7.0+ (arm64-v8a, armeabi-v7a)

## Установка

### Windows

Скачайте один из вариантов из [Releases](https://github.com/saveksme/SaveXSpace/releases):

- **SaveXSpace-Setup.exe** — установщик (рекомендуется)
- **SaveXSpace-Portable.zip** — портативная версия (без установки)

### Android

- **app-release.apk** — установочный файл

## Быстрый старт

1. Скачайте и установите приложение
2. Импортируйте подписку (URL, QR-код или файл)
3. Выберите сервер
4. Нажмите кнопку подключения

## Горячие клавиши

- `Ctrl+V` — быстрый импорт подписки из буфера обмена (если в буфере URL)
